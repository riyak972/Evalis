# Models Directory

This directory contains **sample pre-trained machine learning models**
used only for demonstrating Evalis’s fairness evaluation capabilities.

## sample_model.pkl
- Algorithm: Logistic Regression
- Training data: Simulated resume screening dataset
- Purpose: Demo and testing only

⚠️ These models are **not** intended for real-world decision-making
or production use. They exist solely to showcase fairness auditing
and bias detection features in Evalis.

Developers can replace these models with their own trained models
when extending Evalis.
